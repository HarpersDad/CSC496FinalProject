Roll Stats!!!

This site was created with NODE.js, EXPRESS, and MONGOOSE. Using HTML, JavaScript, Python, and MongoDB.

I set out to create a site that allows a person to "roll" stats for a Dungeons and Dragons 5E character. The user can set the character's name, then they may roll the stats for the character. After this is done, the user can submit the character and save it to the database. I am still working on a retrieval system for players who wish to view saved characters.

I would also like to implement a login system that would allow users to save characters under their username. This way when a person logs in and chooses to retrieve a character, only their created characters will appear.
